# Aplikace pohyb pomáhá

## CZ

## Zásady ochrany soukromí
* Aplikace pohyb pomáhá shromažďuje a ukládá data uvedená níže výhradně na serverech Google Ireland Limited umístěných ve Varšavě.
* Do zařízení jsou ukládána pouze data sloužící pro autentizaci a autorizaci uživatele.
* Respektujeme vaše soukromí, data jsou řádně zabezpečena a ochráněna před zneužitím. Nikdo jiný než členové vašeho Strava klubu nemá k datům přístup.

## Osobní data
Osobní data poskytuje Strava.com, slouží pro zajištění základní funkčnosti aplikace a jsou ukládána v rozsahu souhlasu poskytnutého při registraci aplikace.
* Jméno
* Příjmení
* Zobrazované jméno
* Avatar
* Veřejné sportovní aktivity uživatele z aplikace Strava a jejich detaily (jako např. typ aktivity, vzdálenost, čas, provedené změny nebo smazání aktivity)
* Kluby, ve který je uživatel na Strava.com přihlášen

## Statistická data
Statistická data ukládá aplikace během jejího používání a slouží výhradně pro účely vylepšování aplikace.
* Datum a čas přihlášení uživatele do aplikace
* Typ zařízení a prohlížeče, ze kterého uživatel do aplikace přistupuje

## Kontaktní údaje a správa uložených údajů
Na žádost uživatele odeslané vývojáři budou všechna osobní data spojená s uživatelem odstraněna. Vývojáře aplikace je možné kontaktovat prostřednictvím odkazu v obchodě aplikací.
Tyto zásady ochrany soukromí nabývají platnosti a účinnosti dne 16.07.2023.


## EN

## Privacy Policy
* The pohyb pomáhá application collects and stores the data listed below exclusively on Google Ireland Limited servers located in Warsaw.
* Only data used for user authentication and authorization is stored on the device.
* We respect your privacy; data is properly secured and protected against misuse. No one other than the members of your Strava club has access to the data.

## Personal Data
Personal data is provided by Strava.com, serves to ensure the basic functionality of the application, and is stored within the scope of the consent provided during application registration.
* First Name
* Last Name
* Display Name
* Avatar
* User's public sports activities from the Strava app and their details (such as activity type, distance, time, changes made, or activity deletion)
* Clubs in which the user is enrolled on Strava.com

## Statistical Data
Statistical data is stored by the application during its use and serves exclusively for the purpose of improving the application.
* Date and time of the user's login to the application
* Type of device and browser from which the user accesses the application

## Contact Details and Management of Stored Data
Upon a user's request sent to the developer, all personal data associated with the user will be deleted. The application developer can be contacted via the link in the app store.
These privacy policies become valid and effective on July 16, 2023.
